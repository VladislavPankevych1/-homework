Родини шрифтів та їх варіації

В CSS існує п'ять основних універсальних сімейств шрифтів (generic font families), які використовуються для визначення загального стилю тексту, я вам покажу 3 сімейства шрифта і як вони відображаються з прикладом

Serif

  1. Noto Serif:це глобальна колекція шрифтів для письма всіма сучасними та стародавніми мовами. Noto Serif — це модульований («засічковий») дизайн для текстів латинським, кириличним та грецьким шрифтами, який також підходить як додатковий шрифт для інших шрифтів
  2. Gart Serif:це сучасна українська гарнітура із засічками (2024 р.) від дизайнера Віталія Гоцанюка, що поєднує класичні традиції перехідної антикви 18-го століття з сучасними гострими формами. Вона містить 12 накреслень, кирилицю, розширену латиницю та варіативні версії, ідеальні для заголовків і тексту.
  3. Della Respira:це декоративний шрифт із засічками, доступний на Google Fonts. Він має класичний, витончений вигляд, придатний для заголовків та коротких текстів. Хоча основна версія на Google Fonts орієнтована на латиницю, він часто використовується у комбінації з іншими шрифтами для українських проектів.

Sans Serif

  1. Grotesque:це рубані шрифти без зарубок, які чудово підходять для сучасних заголовків, інтерфейсів, веб-дизайну та друкованої продукції.
  2. Neo-Grotesque:це підкатегорія шрифтів без засічок, що з'явилася в середині XX століття (1950-ті рр.). Вони відрізняються від класичних гротесків XIX століття більшою регулярністю, нейтральністю, високою читабельністю та одноповерховою малою літерою «g». Вони ідеально підходять для інтерфейсів, довгих текстів та корпоративних стилів, забезпечуючи чистоту та ясність.
  3. Humanist:Шрифт, що базується на пропорціях рукописного листа. Вони мають найбільший контраст штрихів, виглядають більш "теплими" та каліграфічними серед інших Sans Serif.

Script

  1. Formal Script:Елегантні, схожі на почерк пером.(Great Vibes)
  2. Casual Script:Більш невимушені, схожі на почерк ручкою чи маркером.(Dancing Script)
  3. Elegant/Display Script:Вишукані, часто використовуються для заголовків.(Pacifico)

Розмір шрифтів

CSS-властивість font-size використовується для визначення розміру шрифта тексту в елементі. Вона дозволяє змінювати розмір шрифтів для заголовків, параграфів, кнопок, посилань та інших текстових елементів. Задання відповідного розміру шрифта є важливим для забезпечення комфортного читання та оптимального відображення тексту на веб-сторінці.

Товщина шрифтів

Для зміни товщини (жирності) тексту в CSS використовується властивість font-weight. Вона приймає ключові слова (normal, bold) або числові значення від 100 до 900 (крок 100), де 400 — це стандартна товщина (normal), а 700 — жирна (bold).

Основні значення font-weight:

  1. Thin 100:Приклад як виглядає шрифт товщина 100
  2. ExtraLight 200:Приклад як виглядає шрифт товщина 200
  3. Light 300:Приклад як виглядає шрифт товщина 300
  4. Regular 400:Приклад як виглядає шрифт товщина 400
  5. Medium 500:Приклад як виглядає шрифт товщина 500
  6. SemiBold 600:Приклад як виглядає шрифт товщина 600
  7. Bold 700:Приклад як виглядає шрифт товщина 700
  8. ExtraBold 800:Приклад як виглядає шрифт товщина 800
  9. Black 900:Приклад як виглядає шрифт товщина 900

Інші корисні властивості

font-style

Властивість font-style у мовах вебверстки (CSS) відповідає за накреслення тексту, переважно для створення курсиву.

  1. normal: Стандартне вертикальне накреслення (за замовчуванням).
  2. italic: Курсивне накреслення. Використовує спеціально розроблений дизайнером похилий варіант шрифту.
  3. oblique: Похиле накреслення. Браузер просто нахиляє звичайні літери під кутом (використовується рідше, якщо немає italic).

line-height

line-height— це властивість CSS, яка визначає відстань між базовими лініями тексту. Вона керує міжрядковим інтервалом (інтерліньяжем), дозволяючи тексту виглядати розрідженим або щільним. Задається у пікселях, відсотках, одиницях em, rem або безрозмірним множником (наприклад, 1.5).

Тут просто безсмислове речення для того щоб показати приклад

letter-spacing

letter-spacing в CSS — це властивість, яка визначає відстань (міжбуквений інтервал) між символами в тексті. Вона дозволяє збільшувати або зменшувати простір між літерами, впливаючи на розрядку тексту. Значення може бути в пікселях (), em, rem або інших одиницях.

text-align

Властивість CSS text-align використовується для горизонтального вирівнювання тексту або інших інлайнових елементів всередині блочного елемента.

text-decoration

CSS-властивість text-decoration використовується для оздоблення тексту: додавання ліній під, над текстом або його перекреслення. Вона керує відображенням ліній (underline, overline, line-through, none). У сучасних стандартах (CSS3) це універсальна властивість, що об'єднує лінію (-line), колір (-color) та стиль (-style).

  1. none — видаляє всі лінії оформлення.
  2. overline — додає лінію над текстом.
  3. underline — підкреслює текст.
  4. line-through — перекреслює текст.

text-transform

Властивість text-transform у CSS використовується для керування регістром символів у тексті (великі, малі літери). Вона дозволяє змінити зовнішній вигляд тексту без зміни самого вмісту в HTML-коді.

  1. uppercase: Перетворює всі літери на ВЕЛИКІ.
  2. lowercase: Перетворює всі літери на малі.
  3. capitalize: Робить першу літеру кожного слова ВЕЛИКОЮ.
  4. none: Значення за замовчуванням.

text-indent

Властивість text-indent у CSS встановлює відступ (червоний рядок) для першого рядка блоку тексту, не впливаючи на інші рядки. Підтримує пікселі (px), відсотки (%), em тощо. Значення може бути додатним (відступ справа) або від’ємним (виступ вліво).

word-spacing

word-spacing — це CSS-властивість, яка використовується для збільшення або зменшення відстані (інтервалу) між словами в тексті.

Призначення кольорів

Властивість color у CSS визначає колір тексту елемента. Вона може приймати значення як для однорідного кольору, так і для прозорості. Найчастіше використовують шістнадцятковий код (HEX, наприклад, #FF0000), назви кольорів (red), RGB (rgb(255,0,0)) або HSL. Можна також використовувати прозорість за допомогою rgba()

Шістнадцятковий формат (HEX)

HEX-код — це спосіб представлення кольору за допомогою шістнадцяткової системи числення. Він складається з шести символів (цифр 0-9 та літер A-F), які вказують на інтенсивність червоного (Red), зеленого (Green) та синього (Blue) кольорів.

Як це працює?

Кожна пара символів відповідає за свій колір. Чим вище значення, тим більша інтенсивність. Наприклад, код #FF0000 — це чистий червоний колір.

Формат запису: #RRGGBB

Наприклад, для кольору #FF5733:

  • FF — червоний (максимум, 255)
  • 57 — зелений (87 у десятковій системі)
  • 33 — синій (51 у десятковій системі)

RGB

RGB (Red, Green, Blue) — це адитивна колірна модель, що формує кольори на екранах (монітори, телефони) шляхом змішування червоного, зеленого та синього світла, де кожна компонента має значення від 0 до 255. RGBA (Red, Green, Blue, Alpha) додає четвертий параметр — альфа-канал, що задає прозорість від 0 (повністю прозорий) до 1 (непрозорий).

Основні поняття RGB/RGBA

  • RGB(R, G, B): Приклад: rgb(255, 0, 0) — чистий червоний.
  • RGBA(R, G, B, A): Приклад: rgba(255, 0, 0, 0.5) — напівпрозорий червоний.
  • Діапазон: Значення кольорів — 0-255, альфа-каналу — 0.0-1.0.

HSL/HSLA:

HSL використовує три параметри для задання кольору: тон (hue), насиченість (saturation) і яскравість (lightness). HSLA додає прозорість.

Тіні

Text-shadow: — Задає тінь для тексту.

  1. Перший параметр: горизонтальний зміщення.
  2. Другий параметр: вертикальний зміщення.
  3. Третій параметр (необов'язковий): розмиття.
  4. Четвертий параметр (необов'язковий): колір тіні.
